# Quick Setup Guide

## Step 1: Get your X (Twitter) keys
1. Go to https://developer.twitter.com
2. Create a free developer account and new app (Project & App)
3. Get these 4 keys:
   - API Key
   - API Secret
   - Access Token
   - Access Token Secret

## Step 2: Email setup (Gmail example)
1. Turn on 2-step verification in Gmail
2. Create an "App Password" for this tool
3. Use that app password in the code

## Step 3: Edit the code
Open main.py and replace the YOUR_... lines with your real keys and email list.

## Step 4: Run
pip install -r requirements.txt
python main.py

The app window opens. Type, paste link, hit Send.
